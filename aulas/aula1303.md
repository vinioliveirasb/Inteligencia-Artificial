# 📌 1. Problemas de Busca em Inteligência Artificial

Em **Inteligência Artificial**, muitos problemas podem ser resolvidos utilizando **métodos de busca**.

A ideia é encontrar um **caminho entre um estado inicial e um estado objetivo**, explorando diferentes possibilidades dentro de um **espaço de estados**.

### Conceitos importantes

- 🟢 **Estado** → situação atual do problema  
- 🚩 **Estado inicial** → ponto de partida  
- 🎯 **Estado objetivo** → solução desejada  
- 🔗 **Ações** → movimentos possíveis entre estados  
- 💰 **Custo** → valor associado ao caminho  

---

# 🔎 2. Tipos de Busca

Os algoritmos de busca podem ser divididos em dois grandes grupos:

## 1️⃣ Busca Cega (Força Bruta)

Também chamada de **busca não informada**.

O algoritmo **não possui nenhuma informação sobre qual caminho é melhor**, apenas explora os nós possíveis.

Características:

- não usa heurística
- testa possibilidades até encontrar a solução
- pode explorar muitos caminhos desnecessários

### Estrutura usada

- 📚 **Pilha (Stack)**  
- geralmente usando **recursão do Sistema Operacional**

Também é necessário manter uma lista de:

- ✅ **Visitados**

Isso evita que o algoritmo fique **voltando para o mesmo estado**.

---

# 🌳 3. Busca em Profundidade (DFS)

A **busca em profundidade** explora um caminho **até o máximo possível** antes de voltar.

### Funcionamento

- usa **pilha**
- segue um caminho até não ter mais opções
- depois **retrocede (backtracking)**

### Estrutura

```
Pilha
+
Lista de visitados
```

### Vantagens

- usa pouca memória

### Desvantagens

- pode demorar muito
- pode entrar em caminhos ruins

---

# 🌐 4. Busca em Largura (BFS)

Também chamada de **busca em amplitude**.

Ela explora **todos os nós de um nível antes de ir para o próximo nível**.

### Estrutura usada

- 📥 **Fila (Queue)**

```
Fila
+
Lista de visitados
```

### Característica

Cria uma **árvore completa de possibilidades**.

### Vantagens

- encontra a **solução mais curta em número de passos**

### Desvantagens

- usa **muita memória**

---

# 💡 5. Busca Heurística (Busca Informada)

Diferente da busca cega, a **busca heurística usa informações extras** para guiar a busca.

Essas informações são chamadas de:

```
Heurística
```

Uma **heurística é uma dica ou estimativa** que ajuda o algoritmo a escolher caminhos melhores.

---

# 💰 6. Custo Real

Representado por:

```
g(n)
```

Significa:

> custo real do caminho percorrido desde o início até o nó atual.

Exemplo:

Se o caminho foi:

```
Arad → Sibiu → Fagaras
```

O **g(n)** será a soma das distâncias percorridas.

---

# ⛰️ 7. Subida de Encosta (Hill Climbing)

Algoritmo heurístico que funciona como **subir uma montanha**.

A ideia é sempre escolher o **vizinho que parece melhor**.

### Funcionamento

1️⃣ avalia os estados vizinhos  
2️⃣ escolhe o melhor deles  
3️⃣ repete o processo  

### Características

- usa **profundidade**
- pode usar **pilha recursiva**

### Problema

Pode ficar preso em:

- ❗ **máximo local**
- ❗ solução não ótima

---

# 🗺️ 8. Exemplo Clássico — Mapa da Romênia

Um dos exemplos mais famosos em IA é o **problema de encontrar um caminho entre cidades da Romênia**.

Exemplo:

```
Arad → Bucharest
```

Cada cidade possui **conexões com outras cidades** e **distâncias associadas**.

Exemplo de caminhos possíveis:

```
Arad → Sibiu → Fagaras → Bucharest
```

ou

```
Arad → Sibiu → Rimnicu Vilcea → Pitesti → Bucharest
```

Cada estrada possui um **custo (distância)**.

# 📊 9. Comparação dos Métodos de Busca

| Método | Estrutura | Informação |
|------|------|------|
Busca em Profundidade | Pilha | Não informada |
Busca em Largura | Fila | Não informada |
Busca Heurística | Varia | Usa heurística |
Hill Climbing | Profundidade | Heurística |

---

# 🎯 Conclusão

Os **métodos de busca** são fundamentais em Inteligência Artificial.

Eles permitem que um agente:

- explore possibilidades
- encontre soluções
- minimize custos

A escolha do algoritmo depende de:

- memória disponível
- tamanho do problema
- existência de heurística
